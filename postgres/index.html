ORA-01578: Data Corruption Index Fix

Problem: ORA-01578: data block corrupted (file # 103, block # 508084)

Solution:

SELECT PARTITION_NAME, SEGMENT_NAME, SEGMENT_TYPE, TABLESPACE_NAME, OWNER
FROM DBA_EXTENTS 
WHERE FILE_ID = 103 AND 508084 BETWEEN BLOCK_ID AND BLOCK_ID + BLOCKS - 1;

ALTER INDEX IGULER.IDX_LOGINC REBUILD PARTITION SYS_P68716 LOGGING ONLINE PARALLEL 8;

ORA-00001: Unique Constraint Violation

Problem: ORA-00001: unique constraint violated

Solution: Check for duplicate values and ensure the column is meant to be unique.
